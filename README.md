# Sales Performance Dashboard

## Project Overview

Developed an interactive Sales Performance Dashboard using Power BI to analyze sales, profit, customer segments, payment methods, and regional performance.

## Tools Used

* Power BI
* Power Query
* Microsoft Excel

## Dashboard Features

* KPI Cards (Sales, Orders, Profit, Ship Days)
* Sales by Category
* Sales by Segment
* Sales by Payment Mode
* State-wise Sales Analysis
* Monthly Sales Trend
* Monthly Profit Trend
* 15-Day Sales Forecast

## Key Insights

* Cash on Delivery was the most preferred payment method.
* Office Supplies generated the highest sales.
* Consumer segment contributed the largest share of sales.
* Sales peaked during September and December 2020.
* Profit was highest during March and October.
